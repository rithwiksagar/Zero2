## Installation

Install and start using AI components in your Next.js project in just a few steps.

<div className="prose dark:prose-invert prose-pre:bg-neutral-800 prose-pre:rounded-md max-w-60">
```bash
$ npx brainframe init
```
</div>

### Requirements

Before installing the library, make sure your project includes:

- *Next.js 14+*
- *React 18+*
- *Tailwind CSS*

If you don't have Tailwind installed yet, follow the Tailwind CSS installation guide for Next.js.
